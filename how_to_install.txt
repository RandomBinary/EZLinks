Installation instructions

1. Set the configuration in the config.php file.

2. Go to the database that you set in config.php using PHPMyAdmin.

3. In PHPMyAdmin, go to the "Import" tab, and import the "urls.sql" file.

4. (OPTIONAL) Edit the "index.php" file to your likings, you may use a free template, just remember to have the form on that page! ;)

5. Finally, upload all the files to your web server.
